## Works on

| Works  | Version          | Notes                                                                                    |
| ------ | ---------------- | ---------------------------------------------------------------------------------------- |
|  Yes   | Windows          | Tested on Windows 11                                                                     |
|  Yes   | Linux (AppImage) | Tested on Fedora (KDE) and CachyOS (Gnome)                                               |
|  No    | Docker           | Not supported since the classes and ids of various elements are different                |
|  No    | Browser Version  | Not supported since the classes and ids of various elements are different                |

## How to install

1. Copy all the contents of the [index.css](https://raw.githubusercontent.com/ZortexSenpai/fluentshin/refs/heads/main/index.css)
2. Paste the contents in the Feishin custom CSS settings. You can find it under: `Settings` -> `Ã€dvanced` -> `Custom CSS`

## Versioning
Major.Minor.Fix
- Major -> Major changes to the skin
- Minor -> Smaller changes / features (for example a single page)
- Fix -> Smaller fixes to existing elements
